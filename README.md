# How to Run — AI Food Genius

Follow these steps on any machine (Windows, Mac, or Linux) to install and launch the application from scratch.

---

## Requirements

- **Python 3.9 or higher**
  Check your version: `python --version`
  Download from: https://www.python.org/downloads/

- **pip** (comes with Python)
  Check: `pip --version`

---

## Step 1 — Download the Project

If you received the project as a ZIP file, extract it to a folder of your choice.

If you are cloning from a repository:
```
git clone <repository-url>
cd food_suggest
```

---

## Step 2 — Install Dependencies

Open a terminal (Command Prompt, PowerShell, or Terminal) and navigate to the project folder:

```
cd path/to/food_suggest
```

Install all required packages:

```
pip install -r requirements.txt
```

This installs: Streamlit, Pandas, NumPy, scikit-learn, XGBoost, LightGBM, CatBoost, and Plotly.

> **Note:** CatBoost and LightGBM are large packages. Installation may take 2–5 minutes depending on your internet speed.

---

## Step 3 — Generate the Dataset

The AI models train on a local dataset. Run this once to generate it:

```
python h1.py
```

This creates the following files in the project folder:
- `final_dataset.csv` — main training dataset (3,000 rows, 26 columns)
- `recipes.csv` — 250 unique recipes
- `user_profiles.csv` — 200 user profiles
- `user_feedback.csv` — user–recipe interaction records
- `nutrition.csv` — nutritional data

> **Skip this step** if `final_dataset.csv` already exists in the folder.

---

## Step 4 — Launch the Application

```
streamlit run app_simple.py
```

The app will open automatically in your default browser at:
```
http://localhost:8501
```

If the browser does not open automatically, copy and paste that URL manually.

---

## First Launch Notes

- The first time you run the app, it will **train 5 AI models** on startup. This takes approximately **30–60 seconds** and only happens once per session (models are cached after that).
- You will see a loading spinner during this step — this is expected.
- A local database file (`food_genius.db`) will be created automatically to store user accounts, ratings, and favourites.

---

## Using the App

1. **Register** a new account using the sidebar (top of the left panel).
2. Go to **AI Recommendations** and fill in your profile (age, diet type, cuisine, health condition, taste).
3. Click **Get My Recommendations** to see your personalized recipe suggestions.
4. **Rate recipes** (1–5 stars) to improve future recommendations via collaborative filtering.
5. Click the heart button on any recipe to save it to **Favourites**.

---

## Generating the PowerPoint Presentation (Optional)

To regenerate the project presentation:

```
pip install python-pptx matplotlib
python generate_ppt.py
```

This creates `AI_Food_Genius_Presentation.pptx` in the project folder.

---

## Troubleshooting

| Problem | Solution |
|---|---|
| `ModuleNotFoundError` for any package | Run `pip install -r requirements.txt` again |
| `streamlit: command not found` | Run `python -m streamlit run app_simple.py` instead |
| App opens but shows errors on startup | Make sure `final_dataset.csv` exists — run `python h1.py` first |
| Port 8501 already in use | Run `streamlit run app_simple.py --server.port 8502` |
| Slow first load | Normal — models train on first run, then cache for the rest of the session |
